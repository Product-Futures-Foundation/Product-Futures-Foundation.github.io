== Remixing Social Production
1. The GNU GPL: Users own and control the physical sources of production.
2. Time Banks: Users gain sources when they commit to work in the future.
3. The Land Value Tax (LVT): Users gain sources when they pay profit.
4. Cell Structure: Subgroups split sources when realistically divisible.
