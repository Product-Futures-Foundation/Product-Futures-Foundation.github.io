=== Value Forms

Fundamental value such as land, water, plants, animals create most of

Use value is the usefulness of a product the end user.

Exchange value is the difference between the price a user pays *now* and the costs an owner paid *in the past*.

    profit = user_price - owner_costs

When user is owner, there is no sale and so profit is undefined.

Future Value is the future productive potential of Sources plus Labor.

Labor Value is the productive potential of human effort (work + metawork (automation, permaculture)) applied to Sources.
