Sources are all the material and immaterial inputs required for production.

Material Sources are Finite things such as land, water, minerals, gases; and also the physical limitations when sharing, such as the electromagnetic spectrum, the *lack of* all pollutions, externalities, etc.

Immaterial Sources are Infinite things such as computer source code (use of a computer language to describe an app), computer object code (the app), genetics, essentially all 'design'.
